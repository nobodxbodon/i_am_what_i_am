## 十二月

### 2日

该片在广州举行路演活动

### 3日

该片在深圳举办了路演

### 5日

该片在武汉举行映后见面会

### 6日

该片在郑州进行路演

### 10日

该片在重庆英皇电影城举行路演，以及国家版权局公布2021年度第十五批重点作品版权保护预警名单

### 11日

该片发布“为自己而战”终极预告

### 14日

发布终极海报，也被网传为原画：

![终极](图/2021_12_14_终极海报.jpg)
